# Generate-Revenue-to-Support-Further-Development

本工程是要求把粮食制作的工艺、配方、步骤像代码一样解构，以开源方式贡献。还包括部分涉及到的设备开源、烹饪机器人控制等。验证结果以开源者做命名之后，整个流程在社区中分享形成标准化，为未来天体登陆爱好者开设连锁化的太空食品餐厅做筹备。

最终目标是在MAILTOPIA地线系统触及的星球上开设餐厅、为途经的飞船提供补给和修补，获得米其林级别的推荐。
